# GitOps AI Platform

This directory contains the initial scaffold for the AI-powered GitOps deployment platform.

## Structure

- cmd/: service entrypoints for detector, manifest-generator, risk-reviewer, and health-watcher
- pkg/: shared platform logic for AI, Kubernetes, GitOps, and policy integration
- web/: React dashboard
- deploy/: Helm charts and manifests
- docs/: project documentation
- examples/: sample application demo
